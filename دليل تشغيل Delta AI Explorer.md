# 🚀 دليل تشغيل مشروع Delta AI Explorer على VS Code

هذا الدليل يشرح الخطوات اللازمة لإعداد بيئة العمل وتشغيل تطبيق مستكشف التخصصات الخاص بجامعة الدلتا على جهازك الشخصي باستخدام Visual Studio Code.

## 📌 المتطلبات الأساسية (قبل البدء)

تأكد من وجود البرامج التالية على جهازك:

1. **Python:** تأكد من تثبيت لغة بايثون (الإصدار 3.8 أو أحدث). _(أثناء التثبيت، تأكد من وضع علامة (صح) على خيار `Add Python to PATH`)_.
    
2. **Visual Studio Code (VS Code):** محرر الأكواد الذي سنعمل عليه.
    
3. **إضافة Python في VS Code:** يُفضل تثبيت إضافة (Extension) الخاصة بـ Python داخل VS Code لضمان قراءة الكود بشكل سليم.
    

---

## 📁 1. ترتيب الملفات وتنظيمها

لضمان عمل الكود بشكل صحيح وبدون أخطاء في قراءة البيانات، يجب تنظيم الملفات كالتالي:

1. قم بإنشاء مجلد (Folder) جديد على جهازك وسمّه اسماً واضحاً باللغة الإنجليزية، مثلاً: `Delta_AI_Project`.
    
2. ضع داخل هذا المجلد **ملفين فقط** لا غير:
    
    - `app.py` (ملف الكود البرمجي الذي يحتوي على واجهة Streamlit والخوارزمية).
        
    - `questions.json` (ملف قاعدة البيانات الذي يحتوي على الـ 200 سؤال).
        
3. افتح برنامج VS Code، ثم من القائمة العلوية اختر `File` -> `Open Folder`، واختر المجلد `Delta_AI_Project` الذي قمت بإنشائه.
    

---

## ⚙️ 2. تحميل المكتبات المطلوبة (Dependencies)

يعتمد هذا المشروع على مكتبات خارجية يجب تحميلها أولاً.

1. داخل برنامج VS Code، قم بفتح موجه الأوامر (Terminal) من خلال القائمة العلوية: `Terminal` -> `New Terminal`.
    
2. تأكد أن المسار الظاهر في الـ Terminal هو مسار مجلد المشروع الخاص بك.
    
3. انسخ الأمر التالي والصقه في الـ Terminal ثم اضغط `Enter`:
    

Bash

```
pip install streamlit plotly
```

_(انتظر بضع ثوانٍ حتى ينتهي النظام من تحميل وتثبيت المكتبات)._

---

## ▶️ 3. كيفية تشغيل الكود

بعد التأكد من ترتيب الملفات وتثبيت المكتبات، اتبع الآتي لتشغيل الموقع:

1. في نفس الـ Terminal داخل VS Code، اكتب الأمر التالي واضغط `Enter`:
    

Bash

```
streamlit run app.py
```

2. **ماذا سيحدث الآن؟**
    
    - سيقوم VS Code بتشغيل خادم محلي (Local Server).
        
    - سيُفتح متصفح الإنترنت الافتراضي لديك تلقائياً على الرابط `http://localhost:8501`.
        
    - ستظهر لك شاشة الترحيب الخاصة بتطبيق **Delta AI Explorer**، وهو الآن جاهز للاستخدام!
        

---

## ⚠️ تعليمات وتحذيرات هامة

- 🛑 **تحذير أسماء الملفات:** لا تقم بتغيير اسم ملف `questions.json` أبداً. الكود في `app.py` مبرمج للبحث عن هذا الاسم بالتحديد. إذا قمت بتغييره، سيتعطل الموقع ويظهر خطأ.
    
- 🛑 **تعديل ملف JSON:** إذا قررت في المستقبل فتح ملف `questions.json` لتعديل سؤال أو وزن معين، كن حذراً جداً ألا تحذف أي فاصلة (`,`) أو قوس (`}` أو `]`)، لأن أي خطأ إملائي في هيكل الـ JSON سيمنع التطبيق من العمل.
    
- 💡 **إيقاف تشغيل الموقع:** عندما تنتهي من تجربة الموقع وتريد إغلاقه، ارجع إلى شاشة الـ Terminal في VS Code واضغط على اختصار `Ctrl + C` من لوحة المفاتيح لإيقاف الخادم المحلي.
    
- 💡 **خطأ "Streamlit is not recognized":** إذا ظهر لك هذا الخطأ عند محاولة تشغيل الكود، فهذا يعني غالباً أنك نسيت تفعيل خيار `Add Python to PATH` أثناء تثبيت بايثون على جهازك. ستحتاج إلى إضافته يدوياً في إعدادات الويندوز أو إعادة تثبيت بايثون مع تفعيل الخيار.
